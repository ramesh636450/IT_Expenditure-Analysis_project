# IT Expenditure Analysis — Power BI

## Overview
Power BI dashboard tracking IT spend, budget variance, and departmental
expenditure trends to support IT financial planning and decision-making.

## Problem Statement
IT teams lacked a consolidated view of budget utilization
and cost distribution across departments and projects.

## Tools Used
- Power BI (DAX, Power Query, KPI cards, trend visuals)

## Key Findings
- Identified overspend categories and budget variance by department
- Built monthly trend tracking for IT cost monitoring
- Created executive-level summary view for leadership reporting

## How to Use
Download the .pbix file and open in Power BI Desktop.
